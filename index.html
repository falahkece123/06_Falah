Selamat Datang di DRY App 👋

Aplikasi demo yang menampilkan 9 data anggota tim menggunakan prinsip DRY, Reuse, Modular, dan Scalable. Semua tampilan dirender dari satu sumber data.

Prinsip Kode

  • DRY — data ditulis sekali
  • Reuse — fungsi render dipakai ulang
  • Modular — tiap komponen mandiri
  • Scalable — mudah ditambah data baru

Komponen Tersedia

  • Array — sumber data tunggal
  • Cards — tampilan kartu anggota
  • List — daftar nama & divisi
  • Menu — navigasi nama
  • Table — tabel lengkap dengan status

Anggota Terbaru

Array Data

9 data dirender dari 1 array — prinsip DRY, Reuse, Modular, Scalable

Prinsip yang diterapkan
DRY — data hanya ditulis sekali di array.  |  Reuse — setiap fungsi render dipakai ulang via data.map().  |  Modular — setiap tampilan terpisah, bisa diubah mandiri.  |  Scalable — tambah data ke-10 cukup tambah 1 baris di array, semua tampilan otomatis ikut.
